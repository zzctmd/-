## 📦 部署前准备：构建项目

### 第 0 步：打包项目文件
在部署前，需要先将项目打包成可部署的静态文件：

1. 打开终端/命令行
2. 进入项目文件夹：
```bash
cd choujiang
```
3. 运行打包命令：
```bash
npm run build
```
4. 等待完成后，会生成一个 `dist` 文件夹

## 🚀 方法一：拖拽上传（最简单）

### 第 1 步：注册 Vercel
1. 打开浏览器访问：https://vercel.com
2. 点击右上角 **"Sign Up"**（注册）
3. 选择用 **GitHub** / **Google** / **Email** 注册
4. 验证邮箱

### 第 2 步：部署项目
1. 登录后，点击 **"Add New..."** → **"Project"**
2. 在页面底部找到 **"or, Deploy a Template"**，往下滚动
3. 或者直接访问：https://vercel.com/new
4. 找到项目文件夹中的 **`dist` 文件夹**
5. 把整个 **dist 文件夹** 拖进浏览器窗口
6. 等待 10-30 秒上传和部署

### 第 3 步：获取网址
部署完成后，会显示：
```
✅ https://choujiang-xxxx.vercel.app
```

**这就是你的抽奖网址！** 发给任何人都能用。

## 🔧 方法二：使用 GitHub（推荐长期使用）

### 优势
- 修改代码后自动重新部署
- 更容易管理和维护
- 有版本控制

### 步骤
1. 将项目代码上传到 GitHub（创建一个新仓库）
2. 登录 Vercel.com
3. 点击 **"Import Project"** 或 **"Add New..."** → **"Project"**
4. 选择 **"Import Git Repository"**
5. 授权 Vercel 访问你的 GitHub
6. 选择抽奖项目的仓库
7. 配置构建设置：
   - **Framework Preset**: Vite
   - **Build Command**: `npm run build`
   - **Output Directory**: `dist`
8. 点击 **"Deploy"**
9. 等待部署完成，获取网址

## 🔄 如何更新

### 如果使用拖拽上传方式
1. 修改项目代码
2. 重新运行 `npm run build` 生成新的 dist 文件夹
3. 在 Vercel 网站找到项目，点击 **"Redeploy"**
4. 或者删除旧项目，重新拖拽上传新的 dist 文件夹

### 如果使用 GitHub 方式（推荐）
1. 修改代码
2. 提交并推送到 GitHub：
```bash
git add .
git commit -m "更新抽奖系统"
git push
```
3. Vercel 会**自动检测并重新部署**，无需手动操作！

### 使用命令行（高级用户）
```bash
# 1. 安装 Vercel CLI（只需一次）
npm install -g vercel

# 2. 登录（只需一次）
vercel login

# 3. 在项目根目录运行
cd choujiang
npm run build
vercel --prod
```

## 📌 小技巧

### 自定义网址（可选）
默认网址比较长，可以：
1. 在 Vercel 项目设置中改名字
2. 或使用短链接服务（如 bit.ly）

### 分享给用户
做一个二维码：
1. 访问 https://www.qrcode-monkey.com
2. 输入你的 Vercel 网址
3. 生成二维码，打印出来
4. 用户扫码即可打开
