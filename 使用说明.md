# 年会抽奖系统 - 使用说明

## 📱 在线使用（推荐）

**网址：** [即将部署后提供]

### 使用步骤
1. 在电脑浏览器打开网址（推荐 Chrome 或 Edge）
2. 准备 Excel 文件：
   - 第一列：姓名
   - 第二列：部门（可选）
3. 上传 Excel 文件
4. 设置奖项名称和人数
5. 点击"开始抽奖"
6. 全屏显示（按 F11）
7. 用空格键控制抽奖

### 注意事项
- 建议在大屏幕或投影仪上使用
- 确保网络连接稳定
- 抽奖数据仅保存在本地浏览器，不会上传
