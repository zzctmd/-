# 🎉 年会抽奖系统

一个现代化的年会抽奖系统，支持 Excel 导入名单，带有炫酷的滚动效果和中奖展示。

## ✨ 特性

- 📊 支持 Excel 导入参与人员名单
- 🎯 自定义奖项名称和获奖人数
- 🎨 精美的动画效果和现代化 UI
- 🔒 自动去重，已中奖者不会重复中奖
- 💾 本地存储，数据安全
- 📱 响应式设计，适配多种屏幕
- ⌨️ 空格键控制抽奖流程

```bash

```

## 📖 使用说明

1. 准备 Excel 文件（第一列：姓名，第二列：部门）
2. 打开系统，上传 Excel 文件
3. 设置奖项名称和获奖人数
4. 按 F11 全屏显示
5. 使用空格键控制抽奖
   - 第一次按空格：开始滚动
   - 第二次按空格：停止并显示中奖者
   - 第三次按空格：进入下一个奖项
