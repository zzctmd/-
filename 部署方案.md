# 抽奖系统部署方案

## 方案对比

| 方案 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| **在线版（Vercel）** | ✅ 无需安装<br>✅ 自动更新<br>✅ 任何设备可用 | ❌ 需要网络 | 有网络的场合 |
| **本地 Windows 程序** | ✅ 无需网络<br>✅ 更稳定<br>✅ 数据安全 | ❌ 需要下载<br>❌ 更新需重新下载 | 无网络或高度保密 |

---

## 🌐 方案一：在线版（推荐）

### 步骤1：打包项目
```bash
cd /Users/zzcnb123456/Desktop/抽奖机器/choujiang
npm run build
```

### 步骤2：部署到 Vercel（免费）

#### 方法 A：使用 GitHub + Vercel（推荐）
1. 在 GitHub 创建仓库
2. 推送代码到 GitHub
3. 登录 vercel.com
4. 点击 "Import Project"
5. 选择 GitHub 仓库
6. 自动部署，获得网址

#### 方法 B：直接拖拽部署
1. 登录 vercel.com
2. 拖拽 `dist` 文件夹
3. 获得网址

**结果：** 得到一个网址，如 `https://choujiang.vercel.app`

## 💻 方案二：Windows 本地程序

### 使用 Tauri 打包（推荐）

#### 步骤1：初始化 Tauri
```bash
cd /Users/zzcnb123456/Desktop/抽奖机器/choujiang
npm install --save-dev @tauri-apps/cli @tauri-apps/api
npm install -g @tauri-apps/cli
```

#### 步骤2：创建 Tauri 配置
创建 `src-tauri` 文件夹，添加配置文件

#### 步骤3：打包
```bash
npm run tauri:build
```

**结果：** 生成 `抽奖系统.exe`（约 5-10MB）

### 交付给用户
1. 把 `.exe` 文件拷贝到 U 盘
2. 用户双击运行即可
3. 附带简单的使用说明 PDF

## 🎯 最简单方案（强烈推荐）

### Electron 快速打包

```bash
# 安装 electron-builder
npm install --save-dev electron electron-builder

# 添加 electron 配置
# 一键打包
npm run electron:build
```

**优点：**
- 生成 Windows 安装包（.exe）
- 双击安装，自动创建桌面图标
- 看起来像专业软件

## 📦 傻瓜式使用包

创建一个文件夹，包含：

```
年会抽奖系统/
├── 抽奖系统.exe          # 主程序
├── 使用说明.pdf          # 图文教程
├── 示例名单.xlsx         # Excel 模板
└── 快速开始.txt          # 简要步骤
```

### 使用说明.txt 内容：
```
【年会抽奖系统 - 使用说明】

1. 双击"抽奖系统.exe"打开程序
2. 点击"上传Excel"，选择名单文件
3. 设置奖项（如：一等奖1人、二等奖3人）
4. 点击"开始抽奖"按钮
5. 按空格键开始滚动
6. 再按空格键停止并显示中奖者
7. 继续按空格键进入下一个奖项

注意：
- 按 F11 可以全屏显示
- Excel 第一列是姓名，第二列是部门
- 中奖者不会重复中奖

技术支持：[您的联系方式]
```

## 我的建议

**对于完全不懂电脑的用户：**

1. **首选：打包成 Windows 程序**
   - 给他们一个 U 盘
   - 里面只有一个 `年会抽奖.exe` 和一个 PDF 说明
   - 双击就能用

2. **备选：在线版**
   - 部署到 Vercel
   - 给他们一个短网址（如：bit.ly/nianhui2025）
   - 打开网址就能用
