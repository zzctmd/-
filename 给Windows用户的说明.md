## 💻 方案二：本地安装版（需要自己打包）

如果您必须使用离线版本，需要在 **Windows 电脑** 上自行打包：

### 📦 打包步骤

#### 第 1 步：安装 Node.js
1. 访问：https://nodejs.org/
2. 下载并安装 LTS 版本（推荐 18.x 或 20.x）
3. 安装时勾选 "Add to PATH"

#### 第 2 步：打包程序
1. 将整个项目文件夹复制到 Windows 电脑
2. 打开命令提示符（CMD）或 PowerShell
3. 进入项目目录：
```bash
cd 项目路径\choujiang
```
4. 安装依赖（首次）：
```bash
npm install
```
5. 打包成 Windows 程序：
```bash
npm run electron:build
```

#### 第 3 步：获取安装包
打包完成后，在以下位置找到安装包：
```
choujiang\dist\大连市自贸区税务局年会抽奖 Setup 1.0.0.exe
```

## 📋 使用教程

### Excel 名单格式
请准备一个 Excel 文件，格式如下：

| 姓名 | 部门 |
|------|------|
| 张三 | 技术部 |
| 李四 | 市场部 |
| 王五 | 财务部 |

**重要**：
- 第一行可以是标题行（会自动跳过）
- 第一列必须是姓名
- 第二列是部门（可选）

项目中包含了 **示例名单.xlsx**，可以参考。

### 抽奖流程
1. 打开系统（在线版或本地版）
2. 点击 **"上传 Excel 文件"**
3. 设置奖项：
   - 奖项名称（如：一等奖、二等奖）
   - 获奖人数（如：1人、3人）
4. 点击 **"🎉 开始抽奖 🎉"**
5. 使用 **空格键** 控制：
   - 第 1 次按空格：开始滚动
   - 第 2 次按空格：停止并显示中奖者
   - 第 3 次按空格：进入下一个奖项
6. 投影时按 **F11** 全屏
7. 抽完所有奖项后，可以返回重新配置

### 注意事项
- ✅ 已中奖者不会重复中奖
- ✅ 数据只保存在本地，不会上传到服务器
- ✅ 关闭程序后数据自动清空
- ⚠️ 建议提前准备好 Excel，并测试一遍流程

## ❓ 常见问题

### Q1: 网址在哪里？
**A**: 部署完成后会提供具体网址，请留意更新。

### Q2: 可以在手机上用吗？
**A**: 可以，但屏幕太小，建议在电脑或平板上使用。

### Q3: 需要联网吗？
**A**: 
- 在线版：需要联网
- 本地安装版：不需要联网

### Q4: 杀毒软件报警怎么办？
**A**: 本地安装版可能被杀毒软件拦截（因为没有代码签名），选择"仍要运行"或"添加信任"即可。推荐使用在线版，不会有这个问题。

### Q5: 能不能修改界面？
**A**: 可以，源码都在项目中，您可以自行修改。需要一定的前端开发知识。
